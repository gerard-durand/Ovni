Fichier :

Ouvrir : Ouvre une bdd : formats .bdd (SDM d'Oktal), .ply (Polygon file de Niratam), .3ds (Autodesk), .obj (Wavefront), .m3d (Milkshape 3D Ascii), .g3d (Format Onera G3D), .off (Object File Format de chez DEC). Quand un fichier a été ouvert, son extension devient celle proposée par défaut.
Ajouter une Bdd : Pour ajouter à la bdd courante avec un autre fichier de bdd. Les objets restent indépendants. Il existe une fonction de fusion de plusieurs objets accessible via le menu (ou l'icône) de "Sélection et déplacement".
Enregistrer : Enregistre avec le nom courant (seulement en .bdd, format Oktal).
Enregistrer sous... : Enregistre sous un nouveau nom (.bdd par défaut). On peut choisir un autre format comme .off, .obj ou .g3d.
Propriétés : Affiche le nombre d'objets, de points, de facettes, d'arêtes, de groupes de toute la Bdd, quelques statistiques de nombres de points par facette ainsi que les dimensions de la boîte englobante.
Préférences : Permet de modifier/ajuster certains paramètres. Ces valeurs sont stockées dans un fichier Ovni.ini et réutilisées au lancement suivant.
L'option de gestion d'antialiasing par OpenGL est utile si la carte graphique ne gère pas seule par hardware cette fonction, via le driver. Fonctionne correctement sur des segments dans tous les cas. Sur les bords de polygones, le fonctionnement n'écessite que le driver de la carte graphique supporte l'option GL_MULTISAMPLE d'OpenGL.
Hardware 3D : Affiche les caractéristiques du Hardware (carte graphique) supportant OpenGL.
Quitter : Pour sortir d'Ovni.

Affichage :

Points / Filaire / Plein : Pour sélectionner le mode de représentation. Dans cette version le mode Plein est actif par défaut. Ces fonctions sont directement accessibles via des icônes de la barre de menus.
Axes et Boîte englobante : Pour afficher ou supprimer l'affichage des axes et/ou de la boîte englobante.Ces fonctions sont directement accessibles via des icônes de la barre de menus.
Source de lumière : Pour matérialiser la position de la source de lumière. Cette fonction est directement accessible via une icône de la barre de menus.
Origine : vue originale.
Vue de face :        Projection X-Z, correspond à une vue de face en standard dans Crira
Vue de profil :      Projection Y-Z, correspond à une vue de profil en standard dans Crira
Vue de dessus Z : Projection X-Y, correspond à une vue de dessus en standard dans Crira
Position observateur : permet d'entrer la position angulaire en Azimut et en Site de l'observateur. Cette fonction force l'angle de rotation en Y à 0.
Zoom Spécifique : En plus de la position angulaire, permet d'entrer une distance de visée et un angle de champ de vue (°). La donnée de distance est exprimée en multiple de la diagonale de la boîte englobante.
Centre de Rotation : Modification manuelle du centre de rotation de la visualisation.
Position de la source : Modification manuelle de la position de la source de lumière (en coordonnées homogènes x,y,z,w : rappel : 0<=w<=1, w=0 => infini, w=1 => x,y,z valeurs réelles).

Primitive :

Construction d'objets simples (cône, cube, cylindre, ellipsoïde, facette, sphère, icosaèdre...)
Note : un icosaèdre est composé de triangles. Chaque triangle peut être subdivisé en 4 sous-triangles. Les sommets intermédiaires créés au milieu de chaque arête sont alors ramenés sur une sphère. On obtient ainsi une sphère plus homogène que via la sphère de base, utilisant des méridiens et des parallèles. Le nombre de subdivisions est limité à 8 car au-delà le nombre de facettes total devient trop important, long à créer par une procédure récurssive et difficile à gérer.
Les primitives créées peuvent être supprimées une à une dans l'ordre inverse de leur création.

Repérage :

Point : Pour repérer un point particulier par une étoile. Les coordonnées du point sélectionné s'affichent. Il est possible de les modifier en cochant la case à droite de la valeur à changer pour activer les entrées. Modifier alors la valeur numérique et cliquer sur le bouton "Modifier" pour effectuer le changement. Le bouton "Undo" devient alors actif pour permettre de revenir en arrière.

Facette : Pour repérer une facette particulière par une couleur
    Il est possible alors de visualiser et parcourir les sommets de la facette, d'inverser la normale, de faire une permutation circulaire des sommets (utile pour certains calculs de normales : facettes très allongées...), de masquer cette facette (la bdd n'est pas modifiée) ou de la supprimer (la bdd est alors modifiée)
    Note : Si la fenêtre Outils/Sélection et déplacement est ouverte, les indications sont synchronisées.
Groupe : Pour repérer un groupe de facettes par une couleur. Si cette fonction est utilisée simultanément avec l'Outil "Sélection et Déplacement", la couleur du groupe sélectionné reste affichée pendant certaines opérations de sélection de facettes : permet ainsi de construire plus facilement un groupe donné .
Matériau : Pour repérer un groupe de facettes du même matériau par une couleur (n'est valable que pour certaines bdd possédant le mot clé correspondant. Sinon Groupe <=> Matériau
Objet : Pour repérer un objet particulier par une couleur
Sens des Normales : Swith ON/OFF colorisant en bleu des facettes "correctement orientées" et en rouge les facettes dont la normale est à l'envers [en fait c'est le sens de parcours des points qui est soit dans les sens des aiguilles d'une montre (GL_CW et alors colorisé en rouge) soit dans le sens contraire ou trigonométrique (GL_CCW et alors colorisé en bleu)]. Accessible aussi via un raccourci clavier (n).

Couleurs groupes : Colorise les facettes par leur numéro de groupe. 
Couleurs matériaux : Colorise les facettes par leur numéro de matériau

Image :

Enregistrer image : Enregistre au format .jpg, .png ou .ppm l'image de la fenêtre OpenGL.


Outils :

Sélection et déplacement : Dialogue interactif permettant de sélectionner soit des points, des facettes, ou un objet. Ce dialogue permet aussi de déplacer la sélection, ou encore d'effectuer, via le bouton "Manipulations de l'objet", la translation, la rotation, la création par symétrie d'un objet ou encore, directement, le masquage ou la suppression d'un objet/facette. Quand plusieurs objets sont sélectionnés, on peut aussi les fusionner en 1 seul onjet (fusion des tableaux de points, facettes, ...). Il peut être utile, mais pas nécessaire, de faire suivre cette action par une simplification de la base de données pour éliminer les doublons de points, normales,...
La plupart des fonctions ont un UNDO, sauf la suppression d'un groupe de facettes
Tout comme dans Repérage, on peut effectuer une permutation circulaire de l'ordre des sommets (utile sur certaines facettes très allongées ou ayant des points alignés ou presque alignés).
La sélection des objets ou des facettes peut être à la souris ou manuelle. Dans ce dernier cas, l'entrée des numéros de facettes se fait dans une fenêtre en entrant le/les numéro(s) au clavier. Pour les numéros de facettes la syntaxe est la suivante :
Les entrées sont séparées par une virgule ou un espace. Si 2 nombres n1 et n2 sont séparés par le signe -, c'est comme si on entrait tous les entiers entre ces 2 nombres (n2>n1).
Exemple : 1 2 3-6 10,15 <=> 1 2 3 4 5 6 10 15
Un bouton permet d'ouvrir directement la fenêtre "Repérage Facette" (plutôt que de passer par les menus). Les indications des 2 fenêtres ouvertes sont alors synchrones.
Cette fonction est directement accessible via une icône de la barre de menus.

Modifications : Dialogue interactif permettant d'apporter des modifications à un objet de la cellule. Il y est possible d'ajouter des points à la cellule soit directement par un clic sur une arête, soit en divisant une arête en plusieurs sous-segments de même longueur.
La fonction "Souder 2 points" peut travailler sur des points du même objet ou sur des points de 2 objets différents. Le premier point cliqué est celui qu'on veut déplacer. Le second point cliqué est celui sur lequel se fait la "soudure".
La fonction "Trianguler la bdd" permet de transformer une base de données avec des facettes quelconques en une base n'ayant que des facettes triangulaires. 3 Méthodes de triangulation sont possibles et proposées via la boîte de dialogue "Préférences" :
La méthode 1 (ou en V) triangule à partir du premier point qui est la pointe commune de chaque V. On obtient ainsi des triangles avec les sommets 1,2,3 puis 1,3,4, puis 1,4,5, ... 1,n-1,n pour une facette à n sommets.
La méthode 2 (ou en Z) triangule en alternant le choix des sommets sur les côtés du polygone définis par la succession des sommets 1,2,3,...ou n, n-1, n-2... Les facettes sont définies par les points 1,2,3 puis 1,3,n puis n,3,4, puis n,4,n-1 ...
La méthode 3 (ou en étoile) crée d'abord un point au centre de la facette, puis se sert de ce nouveau point comme point de départ de la triangulation, comme dans la méthode 1. Cette méthode évite l'écueil des points originaux alignés, mais crée beaucoup plus de triangles que les 2 autres méthodes.
Cette fonction est directement accessible via une icône de la barre de menus.

Choix des éléments à afficher : Une boîte de dialogue permet de choisir quels sont les objets à afficher ou masquer
Afficher les élements masqués : Réaffiche tous les éléments masqués, facettes et objets
Restituer les objets supprimés: Restituer tous les objets qui ont été marqués comme étant supprimés de la bdd en mémoire (la suppression n'est effective qu'à l'enregistrement sur fichier)

Transformations :

Ces transformations affectent la base de données entière de l'avion (en mémoire). Il faudra passer par le menu "Enregistrer" pour les sauvegarder.
X => -X : change le signe des coordonnées en X
Y => -Y : change le signe des coordonnées en Y
Z => -Z : change le signe des coordonnées en Z
X <=> Y : interverti les coordonnées X et Y
X <=> Z : interverti les coordonnées X et Z
Y <=> Z : interverti les coordonnées Y et Z
X => Y => Z => X : Permutation circulaire des coordonnées X, Y et Z
Inverser les normales : Inverse toutes les normales de la bdd.
Déplacer : Active un menu pour déplacer la Bdd suivant les 3 axes (Valeurs DX, DY et DZ). Possibilité d'annuler le déplacement précédent en activant le bouton "Inverser".
Facteur d'échelle : Active un menu pour appliquer un facteur d'échelle global sur la bdd. Possibilité d'annuler ce facteur en activant le bouton "Inverser".

Options :

Sous-menu :
Couleurs des groupes : pour modifier les 2 composantes de la lumière (Ambiante et Diffuse) de chacun des groupes
Relire une palette : Relire un fichier de palette de couleurs pour les groupes
Enregistrer la palette : Enregistrer la palette de couleurs courante pour les groupes
Zoom automatique : Calcul d'un Zoom automatique en fonction des coordonnées Min et Max
Centrage automatique : Calcul d'un nouveau centre de rotation (milieu des min et max en X et Y).
