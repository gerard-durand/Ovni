Chill defaults

If type and range checking are set automatically by GDB, they both default to on whenever the working language changes to Chill. This happens regardless of whether you or GDB selected the working language.

If you allow GDB to set the language automatically, then entering code compiled from a file whose name ends with .ch sets the working language to Chill. See Having GDB set the language automatically, for further details.
